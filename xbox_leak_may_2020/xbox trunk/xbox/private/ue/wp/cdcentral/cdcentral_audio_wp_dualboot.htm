Dual Booting a PC
(Windows 9x and Windows 2000)

By Scott Selfon, Audio Content Consultant, Content and Design Team, Xbox Advanced Technology Group

Microsoft Confidential

This is part of a series of "column-style" white papers on various aspects of Xbox audio. If you would like to see a specific area addressed, please send e-mail to content@xbox.com.

Introduction

Microsoft® Windows® 2000 is the development operating system for all aspects of authoring for the Xbox™ video game system from Microsoft, including audio. That said, some tools and audio cards do not yet support Windows 2000. An e-mailed or written request will help move that process along, but for the time being, one workaround to consider is making your machine able to dual boot so that you can boot the machine to either Windows 9x (95, 98, Millennium) or Windows 2000 without needing to reinstall the operating system. This paper presents an overview for how to do this.

Operating System Installation Order

Order of installations is important. It is generally recommended that Windows 2000 be the last operating system installed. If your machine already has Windows 2000 and you want to make it dual boot with Windows 9x, the process is a bit more complicated than the one detailed below. For more information, consult http://support.microsoft.com/support/kb/articles/Q153/7/62.asp.

Formatting Your Hard Drive Partitions

You will need to have either multiple hard drives or multiple partitions on a single hard drive. To repartition a hard drive, you can use the Fdisk and Format tools. As always, when dealing with hard drive formatting, you should back up all of your data, because it will be lost in the process of repartitioning. For more information, consult the Knowledge Base article at http://support.microsoft.com/support/kb/articles/Q255/8/67.ASP.

Windows 2000 supports the Windows NT file system (NTFS), file allocation table (FAT), and FAT32 formatted drives and partitions. Windows 9x supports FAT (Win95) or FAT32 (Win95 OSR2.5, Win98, WinME). Note that Windows 9x does not support NTFS, so you should format all of your partitions to be FAT or FAT32.

Applications on Dual Boot Systems

Any applications that you intend to use on both operating systems will need to be installed once on each operating system; you can't share applications across operating systems.

Setting Up for Dual Boot

Before making any significant system modifications like this, you should back up critical data and make sure that your installation disks for applications, drivers, and so on are handy, just in case.

Once you have two hard drives and/or partitions, pop in the Windows 2000 CD, and click the Install Windows 2000 option in the menu that comes up.

From the menu that comes up, click Install a New Copy of Windows 2000.



Then just follow the installation instructions. You will be prompted for a location to install to. The easiest way to keep track of the two different operating systems is to place this one on your second drive or partition (for instance, D:\WINNT, assuming that your Windows 9x boot drive was C:).

Picking Your Operating System on Each Reboot

After installation is completed, every time you restart the computer, a text boot screen will appear that lets you choose which of the two operating systems you want to run. You can modify the file boot.ini (in Notepad or another text editor) to control how long it takes to time out (before the machine picks the default operating system), what the default operating system is, and more advanced boot options.

For more details on dual booting, please consult specific Knowledge Base articles at http://msdn.microsoft.com.
