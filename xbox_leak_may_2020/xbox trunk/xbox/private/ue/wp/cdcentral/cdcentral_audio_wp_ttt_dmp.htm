Tips, Tricks, and Techniques for DirectMusic Producer

By Chanel Summers, Worldwide Creative Manager, Xbox Content and Design Team

Microsoft Confidential

We have compiled this list of tips, tricks, and techniques to use when working with Microsoft® DirectMusic® Producer, the content creation tool for DirectMusic. We will add more tips and information on a regular basis, so please check this paper for any updates we may have made. If you have any suggestions for more tips, please send e-mail to content@xbox.com.

  1. Use part linking to minimize file size.
    Whenever an element is to be reused as is in styles (that is, a drum groove, comping figure, and so forth), don't copy and paste; instead, create a part link (right click, then click Add Pattern Part(s) from the shortcut menu that appears, then choose Link to Existing Part… from the dialog. Finally, choose the part to link the new part to from the Link to Existing Part list). This has the added benefit of only storing a single copy of the data, so editing one part will alter every other part that is linked to it.


  2. Mix and match extended wave files with short (single note) instrumental samples as the creative circumstance dictates.
    With DirectMusic, it's not an either/or proposition. Where the performance cannot be encapsulated into single samples (such as a sax solo), go ahead and incorporate "mini performances" into the mix as wave tracks or longer Downloadable Sounds (DLS) samples—both options give you the ability to trigger the wave at different speeds/pitches and volumes, as well as to control the wave's playback through MIDI controller curves.


  3. Add a chord track to a sequence track (or wave track) so that interactive elements can be added.
    This is a fairly easy way to get into interactivity—the underlying music remains a static linear sequenced file or an audio file. Secondary segments can then be layered on top and are shaped harmonically by the chord track. Adding a chord track won't modify its sequence track (all notes will still play as authored).


  4. Add a groove track to a segment and author convincing transitions to your project's other pieces of music.
    Every bar might be a different groove, so the music at bar 5 would have a fill of groove 5 to match the melody or intensity of the current music.


  5. Use part locking (available in each part's Properties page—Variations frame, Lock ID drop-down menu) to fatten up thinner sounds or to create flange/delay effects while still maintaining the variability in your performance.


  6. Create secondary segments with volume control changes to interactively shape the mix.
    This can be used to bring in/out certain elements that match the drama of the music. It can also be used to fade smoothly from one texture to another while keeping the form intact.


  7. Gradually combine sound effects into a rhythmic pattern that takes on musical characteristics. (See Bookmark #16, "Motifs in Context," in the DirectMusic Producer 8.0 project for an example of this.)


  8. Create musical textures by combining only secondary segments, each with its own length and time signature.
    The multiple secondary segment toolbar is very handy for auditioning these kinds of segments in Producer.


  9. Create different numbers of variation in each part.
    For example, the drums might have three variations, the bass five, the sitar eight. Different juxtapositions keep the resulting music fresh.


  10. Use the different variation modes for different parts
    For instance:
    • Use the "playing from sequence from first" feature if you want a bass groove that "develops" over variations.
    • If your variations for the flute part tend to overlap and develop in a specific order, try using the "play in sequence from random" feature.
    • For sound effects where you don't want to hear the same version of an effect twice in a row, try using the "no repeats" feature.
    • For very recognizable content (dialog, for instance) in which you might want to make sure the user hears every variation before the content repeats, try using the "shuffle" feature.


  11. Separate the elements of a musical performance into four categories that can be interactively reconstructed: the content (rhythmic and melodic tendencies), the chords, the texture (sparse or full), and the timbres (the instruments).


  12. Use random settings in velocity and start time.


  13. Use inversion boundaries to give the illusion of voice leading for chordal parts.
