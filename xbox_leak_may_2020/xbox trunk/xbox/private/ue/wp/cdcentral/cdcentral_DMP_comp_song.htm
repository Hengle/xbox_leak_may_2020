DirectMusic Producer Components

Song

A Song is a long piece that you create by stringing together several Segments or parts of Segments. You can create transitions to move smoothly from one Segment to the next.

You can use Song Designer to create a long performance that plays multiple Segments in a row. You can organize the Segments to play in any order, repeating a Segment in the order as needed, and using Transitions from one Segment to the next.

You can also define a completely new Segment by placing any number of individual Tracks from different Segments together in a Song.
