3-D 2-D Wood Textures

By Dave McCoy, Graphics Techniques Consultant, Xbox Content and Design Team

The Xbox™ video game console from Microsoft allows for true 3-D textures. Three-D textures use a number of 2-D images and interpolate pixels between the 2-D textures along the third axis. Up to 256 textures can be linked to create a 3-D texture.

However, when creating wood surfaces, a single 2-D texture can be applied using 3-D projection, allowing for a lot of variety without using a lot of texture space.

Everyone has seen the objectionable striping that occurs when a 2-D texture is plane projected through a solid. This simple tip takes advantage of that effect to assist in creating wood surfaces.

We start with a 2-D texture of some wood.



Just about any picture (or painting) of wood will work because of the way the texture is applied. It's better if the texture will tile in all directions as this one does, but it's not strictly necessary depending on how you use the texture. Some high-frequency detail helps because the texture is going to be stretched significantly.

Rather than plane mapping the texture flat onto a surface, project it edge on, but angled a few degrees. This stretches the texture through the length of the object.



Because wood tends to have long, regular markings, the usually objectionable texture stretching with this projection winds up creating the distinctive appearance of wood with very little texture usage. You could just stretch a texture and get the same effect on one side, but a nice side effect of this approach is that the edges of the object show the tight grain of crosscut wood, and all sides of the object look correct and unique with one texture projection.



A definite bonus of this technique is that small changes in the angle and scaling used in projecting the texture produce a unique appearance to the wood surface that doesn't exhibit the usual tiling and resolution problems when trying to use one texture for objects of different scales. All three axes can be varied in scale and rotation, and a lot of variety is possible with just a single texture because the texture intersects the surface uniquely with each change.

Here is the same texture applied to a plank with minor changes to angle and scale.



While the Xbox certainly has sufficient texture memory to allow unique textures to be used to create variety, this approach keeps the number of texture files that need to be referenced and loaded to a minimum, which keeps resource size small and loading times fast. If you feel the need to use more texture memory, just use a really high-resolution wood texture. :^) Naturally, for really high-fidelity wood, the same or a related texture and projection can be used as the basis for bump and specularity maps.

To increase authoring speed, the number of axially aligned objects can be quickly textured with a single projection using this approach. This can speed texturing of multiple objects without significant attention to avoiding texture tiling. Each object appears unique because of the interaction between the surface and the texture. This object was textured with two projections—one near vertical and the other near horizontal.



As a final advantage, this approach uses the same texture again and again without recognizable repetition but minimizes state changes that would occur if different textures were used to create the same variety. State changes can significantly slow drawing speed in high-complexity scenes on high-powered graphic systems.
