Xbox Development Kit for Visual Studio .NET

Features

  • Lets you create, build, and deploy Xbox projects using Visual Studio .Net

Bugs / limitations

  • Requires that you install Visual Studio 6 and the XDK for Visual Studio 6.
  • Currently the Visual Studio .Net debugger is not supported.
    • However, you can use the XDK for Visual Studio 6 to debug Xbox applications compiled with XDK for VS.Net.
  • Some of the Xbox-specific UI property pages aren’t hooked up to the tools. (e.g. the Xbox Deploy Tool’s Additional Files property doesn’t do anything.)
  • Some of the standard property pages allow you to set settings that are inappropriate for the Xbox. (Such as pre-Pentium-III optimization flags.)
  • Does not convert XDK for Visual Studio 6 project files.

Installation

 On a blank machine:

  1. Install the XDK (which requires that you install VS6)
  2. Install the release version of VS.Net (Build 7.0.9466)
  3. Double-click on the VS7XDK.msp file to install the XDK for Visual Studio .Net.

 How to tell if the installation was successful:

  1. Start Visual Studio .Net
  2. From the Start Page, Choose “New Project”
  3. Click on the Visual C++ Projects folder in the “Project Types:” pane.
  4. Click on the XboxWiz icon in the “Templates:” pane.
  5. Enter a name in the Name: field.
  6. Click on OK
  7. You should see an Xbox Wizard Dialog Box appear. Click Finish
  8. Right-click on the project’s icon in the “Solution Explorer” window.
  9. Choose Properties.
  10. Verify that there are two property folders at the bottom of the list of properties: “Xbox Image” and “Xbox Deployment”.
    1. If you don't see these two property folders, you may need to reboot your machine, and create a new project.
      1. If you have to reboot, it's a bug, so please file a bug report.
  11. Press Cancel to close the project property pages.
  12. NOTE: before building for the first time, open a command shell and type the command:
    1. xbdir -x YOURXBOXNAME xe:\
  13. Build and deploy the project by choosing “Build Solution” from the “Build” menu.

Uninstall

To uninstall:

  1. Open the control panel
  2. Choose "Add/Remove Programs"
  3. Choose the "XDK for Visual Studio.net" program from the list of currently installed programs. You may have to scroll down to see this program.
  4. Click on the "Remove" button.
  5. Answer "Yes".
