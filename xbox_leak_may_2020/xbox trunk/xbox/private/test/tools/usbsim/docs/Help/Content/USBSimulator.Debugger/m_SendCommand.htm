SendCommand

Sends a command string through the debug channel on the XBOX.

Parameters
cmd
String specifying the command to send.
CommandDescription
"NOTIFY"Tells the debug channel to start sending responses on the TCP connection. Note that this sends the debuger into a special mode where other commands are ignored.
"REBOOT"
"SYSTIME"
"XBEINFO"
"TITLE"
"PCLIST"
"QUERYPC"
"MODULES"
"MODSECTIONS"
"MODLONG"
"FUNCCALL"
"QUERYKMEMINFO"
"QUERYCPUTIMEINFO"
Remarks
Note that all commands must be terminated with a proper new line character sequence (0x0D 0x0A)
