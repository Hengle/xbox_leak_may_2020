Viewing Debug Output

Steps
  1. With Simulator unit powered down, attach "RUN" end of cable to the RUN COM port.
  2. Configure HyperTerminal or other terminal program connected to the Simulator running at 9600 baud, 8, N, 1, with handshake (flow control) disabled.
  3. Power up the Simulator unit.
