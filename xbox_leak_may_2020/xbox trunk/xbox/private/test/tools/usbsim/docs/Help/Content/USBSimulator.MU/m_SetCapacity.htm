SetCapacity

Configures the size of the memory unit.

Parameters
numBlocks
The number of blocks on the memory unit
blockSize
The number of bytes in each block
logicalSize
Optional parameter, the number of blocks per logical block (default == 0)
Remarks
The default size is 8 megs (SetCapacity(0x3EFF, 0x0200, 0)).
Never change the capacity of a memory unit while it is Pluged in.
Changing the capacity will flush any data currently stored and set the memory to 0xFF (equivelant to the Unformat method).
