Configuring the IP Address

Create an AUTOEXEC.BAT file from the template below, then follow the Downloading Files steps to transfer the file to the simulator.

 WARNING!!!
Your batch file MUST contain this line:
   fileboss.exe /A /B4
Otherwise you will not be able to download any other additional files and depending on your autoexec.bat file you may render the USB Simulator useless.

Sample IP Settings
nets110 IP=DHCP
nets110 IP=157.56.10.19 Mask=255.255.254.0
nets110 IP=209.43.24.188 Mask=255.255.255.0
AUTOEXEC.BAT
