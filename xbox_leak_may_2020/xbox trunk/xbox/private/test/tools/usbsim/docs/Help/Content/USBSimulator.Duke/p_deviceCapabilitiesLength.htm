deviceCapabilitiesLength

This property sets the length of the device's capabilities buffer: input resolution etc.

Parameters
type
String specifying the capability type.
typeDescription
"IN"Device input such as buttons (default is 20 bytes)
"OUT"Device output such as motors (default is 6 bytes)
Remarks
This property is read and write.
