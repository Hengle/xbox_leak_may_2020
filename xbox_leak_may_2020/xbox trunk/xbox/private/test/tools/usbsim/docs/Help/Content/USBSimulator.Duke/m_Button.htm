Button

Sends button press and release information to the host.

Parameters
buttonName
String specifying the name of the button to press.
buttonNameDescription
"A"Analog button A
"B"Analog button B
"X"Analog button X
"Y"Analog button Y
"BLACK"Analog Black button
"WHITE"Analog White button
"LEFTTRIGGER"Analog left trigger
"RIGHTTRIGGER"Analog left trigger
"LEFTTHUMB"Digital button under left thumb stick
"RIGHTTHUMB"Digital button under right thumb stick
"BACK"Digital Back button
"START"Digital Start button
"DPADN"DPAD up
"DPADNE"DPAD up and right
"DPADE"DPAD right
"DPADSE"DPAD down and right
"DPADS"DPAD down
"DPADSW"DPAD down and left
"DPADW"DPAD left
"DPADNW"DPAD up and left
"UP"Left thumb stick up
"DOWN"Left thumb stick down
"LEFT"Left thumb stick left
"RIGHT"Left thumb stick right
Remarks
This method will send the appropriate button state information for buttonDownDelay milliseconds, then send the default (idle) state information for buttonUpDelay milliseconds before returning.

To send multiple buttons at once, or to send different values on the analog buttons, use the SetButtonState method and your own sleeps.
