Exposed Constants

The following tables show the constants values visible through the COM object's type library.

Exposing Constants in Script
To expose the constants in your script, use Windows Scripting Host and the REFERENCE tag:

Logfile Formats
Values passed to Simulator.LogFormat.
Value Description Name
0 Text log file (.txt) RECORDING_FORMAT_TEXT_LOG
1 CATC's Chief UTG format (.utg) RECORDING_FORMAT_CHIEF_UTG
2 XML format (.xml) RECORDING_FORMAT_TEXT_XML

USB Endpoint Modes
Values passed to Device.EndpointMode.
Value Description Name
0 Disables the endpoints ENDPOINT_DISABLE
1 Enables the enpoints for normal operation. Turns off Auto Repeat if its on. ENDPOINT_ENABLE
2 Turns on Auto Repeat. ENDPOINT_AUTOREPEAT

Device Types
Value Description Name
-1 Invalid device type DEVICE_UNKNOWN
0 Duke Controller DEVICE_DUKE
1 Memory Unit DEVICE_MU
2 Game Voice DEVICE_HAWK
3 DVD Remote DEVICE_DVDREMOTE
4 Total number of supported device DEVICE_MAX_TYPES

USB Simulator Error Codes
Internal error codes used by usbsim.lib.
Value Description Name
0 The operation completed successfully. USBSIM_ERROR_OK
1 Device is already connected. USBSIM_ERROR_CONNECTED
2 Device is not connected. USBSIM_ERROR_NOTCONNECTED
3 Use GetLastError to get more information. USBSIM_ERROR_CONNECT_FAILED
4 Use GetLastError to get more information. USBSIM_ERROR_SOCKET_ERROR
5 Parameter is out of range or denotes a port currently not in use. USBSIM_ERROR_INVALID_USB_PORT
6 Specified simulator does not exist. USBSIM_ERROR_INVALID_SIMULATOR
7 Pointer to a USBDevice is not valid. USBSIM_ERROR_INVALID_DEVICE
8 Abort specifed by the user. USBSIM_ERROR_USER_ABORT
9 Timeout waiting for response from simulator. USBSIM_ERROR_TIMEOUT
10 Got data from the simulator but it is likely corrupted. USBSIM_ERROR_DATA_CORRUPTED
11 Use GetLastError to get more information. USBSIM_ERROR_SEND_FAILURE
