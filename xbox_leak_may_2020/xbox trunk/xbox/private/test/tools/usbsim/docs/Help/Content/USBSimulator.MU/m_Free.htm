Free

Returns the storage memory back to the heap.

Parameters
none
Remarks
Because some scripting language (such as JScript) use garbage collection to release unreferenced objects, it is quite possible to have multiple MU objects in memory just waiting around for the garbage collection to free them. To avoid extraneous memory utilization, the Free() method returns this memory immediately back to the heap.

Use this method when you no longer need the object and want to release its resources.
