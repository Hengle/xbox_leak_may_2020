USBSimulator.Simulator

This object provides information and settings on the actual simulator device.

Properties
PropertyDescription
logFilenameSets the name of the log file.
Methods
MethodDescription
GetLastErrorThis method returns the Win32 error code from the last operation.
GetMACAddrThis function returns the MAC address of the simulator. This value can be used to uniquely identify a specific unit.
GetUSBPortsThis function returns a bitmask of which USB Boards are connected and responding.
GetVersionThis function returns the version of the simulator's software.
LogAllows the caller to add an entry to the log file.
LogFormatSets the format of the log file.
