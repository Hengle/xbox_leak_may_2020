Device Enumeration Always Fails

Simulator.GetUSBPorts() returns 0
The USB Simulator requires some initialization after powerup. This initialization occurs when the simulator receives the IPQuery network command. Until this command is recieved, the simulator will not initialize any of it's USB boards (thus GetUSBPorts returns zero).
The Simulator.GetVersion method sends the IPQuery command, so you should always call this method in your script to ensure the USB Simulator is fully ready.
