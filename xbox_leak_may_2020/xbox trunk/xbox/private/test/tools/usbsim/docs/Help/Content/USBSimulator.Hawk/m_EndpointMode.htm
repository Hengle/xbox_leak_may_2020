EndpointMode

Sets the mode of the USB Endpoints on the device.

Parameters
mode
ConstantValueDescription
ENDPOINT_DISABLE0Disable the device
ENDPOINT_ENABLE1Set the device to normal operation (default)
ENDPOINT_AUTOREPEAT2Turns Auto Repeat on (the last sent packet is automatically resent when requested)
Remarks
Note that the above values are exposed through the type library, see the constants page for more information.
