Log Specification

This is the format of the log file when using the default (Simulator.LogFormat(0)).

Format
00052841 010D20C8 [FROM SIM]: // (USB, DATA: IN)
    1, 0, 2, 3
    96 01 02
00052842 010D20C8 [TO SIM]:   // (USB, DATA: DATA1)
    1, 0, 2, 21
    D2 00 14 02 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00

ColorDescription
00052841Time in milliseconds
010D20C8A session wide unique ID for the virtual device
[FROM SIM]Direction of the packet
// (USB, DATA: IN)Description of the packet
1, 0, 2, 3Packet Header (command, subcommand, endpoint, data size)
96 01 02Packet Data. If the header command equals 1 then the first byte is the USB PID
