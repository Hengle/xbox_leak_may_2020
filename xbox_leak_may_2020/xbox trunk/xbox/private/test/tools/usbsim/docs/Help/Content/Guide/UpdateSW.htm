Updating the Software on the Simulator

Steps
  1. With Simulator unit powered down, attach "LOAD" end of the cable to the LOAD COM port.
  2. Configure HyperTerminal or other terminal program connected to the Simulator port at 115200, 8, n, 1 with handshake (flow control) disabled and YMODEM transfer protocol selected.
  3. Power up the Simulator unit. You will see the Fileboss prompt within about a minute.
  4. Make sure that the C drive is selected and enter the 'L' command.
  5. Go to your terminal program file transfer feature and select the file you wish to send to the Simulator.
  6. Repeat steps 4 and 5 for each file you need to transfer.
  7. Disconnect the load cable from COM port and reboot the Simulator.

 WARNING!!!
If you are updating the AUTOEXEC.BAT file, it MUST contain this line:
   fileboss.exe /A /B4
Otherwise you will not be able to download any other additional files and depending on your batch file you may render the USB Simulator useless.
