Setup

Quick Start
  1. Attach the USB Simulator to an Ethernet (10/100 Mbit) HUB
  2. Attach the controller PC to the same HUB
  3. Plug the USB cables into the Simulator and the host
  4. Power on the USB Simulator
  5. Wait for the READY light (takes ~45 seconds to boot)
  6. Run the controller code
Slow Start
  1. Attach all the necessary cables
  2. Follow these directions to set the IP address
  3. Reboot the USB Simulator
  4. Install/register the USBSimulator.dll COM object on your controller PC:
    regsvr32 usbsimulator.dll
  5. Run the controller code (sample).
