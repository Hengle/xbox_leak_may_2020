USBSimulator.Duke

Virtual USB device which mimics the XBOX Duke controller.

Properties
PropertyDescription
buttonDownDelayDefines how long the Button method sends button press information to the host before releasing the button (returning to the default (idle) state).
buttonUpDelayDefines how long the Button method will send the default (idle) state after a button press.
configurationDescriptorThis property sets the USB Configuration Descriptor.
deviceCapabilitiesThis property sets the device's capabilities: input resolution etc.
deviceCapabilitiesLengthThis property sets the length of the device's capabilities buffer
deviceDescriptorThis property sets the USB Device Descriptor.
deviceNameThis read only property returns a description of the device.
enumeratedRead only property returns true when the device has been successfully enumerated.
IPSets or retrieves the IP address of the Simulator.
portSets or retrieves the USB port (1-based index) on the Simulator that this object is assigned to.
recordEnables or disables logging.
xidDescriptorThis property sets the USB XID Descriptor.
xidDescriptorLengthThis property sets the length USB XID Descriptor buffer.
xidPacketThis property sets the user input returned to the host.
xidPacketLengthTThis property sets the length XID packet buffer.
Methods
MethodDescription
ButtonSends button press and release information to the host.
EndpointModeSets the mode of the USB Endpoints on the device.
FreeImmediately releases any resources.
PlugThis method attaches the virtual device to the host, thus initiating the USB enumeration sequence.
SetButtonStateSpecifies the raw state data sent to the host when polled.
UnplugThis method detaches a previously inserted device.
