Description
===========
   The AmplitudeModulation DSP samples shows how to write and build a basic
DSP effect for execution on the Xbox.  This effect takes 2 inputs - the first
is the input signal, the second is an oscillator to be multipled with the input
signal.  The single output is the product of this modulation.
   

Required files and media
========================
   You will need to get the DSP assembler from 
"http://www.metrowerks.com/embedded/suite56/" in order to assemble DSP
effects.
