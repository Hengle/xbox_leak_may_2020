Description
===========
   This sample illustrates Xbox Online account logon and authentication. 
   The sample shows proper online multitasking using XOnlineTaskContinue(),
   as well as error handling.


Programming Notes
=================
   Sample shows how to enumerate accounts on hard disk and memory units.
   For brevity, the sample chooses the first account and controller
   when loggin on.
