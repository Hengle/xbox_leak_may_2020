Description
===========
   The bare bones necessary to complete a DPlay session.
   

Programming Notes
=================
   This is a very stripped down DPlay sample that illustrates the bare
essentials of connecting and transferring data.

   You must set the IP address of the XBox running this sample by editing
basicdplay.cpp and changing the following line:

#define LOCALIP L"xxx.xxx.xxx.xxx"
