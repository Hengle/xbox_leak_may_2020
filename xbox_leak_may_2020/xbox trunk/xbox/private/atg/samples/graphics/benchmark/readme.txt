Description
===========
   The BenchMark sample renders a large number of polygons under various
   conditions to benchmark the performance of the system. A large number of 
   polygons are rendered and various options can be altered, such as the number
   of textures and lights.
   

Required files and media
========================
   Copy the media tree to the target machine before running this sample.


Programming Notes
=================
   There's nothing special about this sample, other than for benchmarking
   purposes.
