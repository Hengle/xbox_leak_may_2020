Description
===========
   The Tiling sample shows how the tiling and z-compression of the rendertarget
   affects performance.
   

Required files and media
========================
   Copy the media tree to the target machine before running this sample.


Programming Notes
=================
   Tiling and z-compression is a very advanced feature of the Xbox graphics
   hardware. Please thoroughly read the documentation and whitepapers to
   understand the effects on your app.
