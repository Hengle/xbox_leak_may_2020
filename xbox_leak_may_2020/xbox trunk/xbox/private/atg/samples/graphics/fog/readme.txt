Description
===========
   The Fog sample illustrates how to use fog in Direct3D. Various controls are
   used to control fog parameters for the 3D scene. 


Required files and media
========================
   Copy the media tree to the target machine before running this sample.


Programming Notes
=================
   Fog in D3D is controlled easily via a few render states. See the FrameMove()
   function for how the render states are used.
