Description
===========
   The ModifyPixelShader sample shows how to use the API to modify a pixel
   shader on the fly.


Required files and media
========================
   Copy the media tree to the target machine before running this sample.


Programming Notes
=================
   The code creates a pixel shader, and then modifies it on the fly.
