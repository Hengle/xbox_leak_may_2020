Description
===========
    Profiles Xbox storage devices for throughput, random access and
    other relevant statistics.


Required files and media
========================
   Copy the media tree to the target machine before running this sample.


Programming Notes
=================
   Generates text files that can be copied from the Xbox to the development
   system and used as Excel input files
