Description
===========
   The DMNotifications sample demonstrates how to use DirectMusic Notification
messages to get information about audio playback.  This sample pays attention
to Segment notifications to find out when segments begin/end playing, and uses
Measure/Beat notifications to find out when a beat occurs in the primary
segment.
   

Required files and media
========================
   Copy the media tree to the target machine before running this sample.
