Description
===========
   The XMOClient sample shows how to use a simple client for an XMediaObject,
   or XMO. The sample simply streams a .wav file from the disk, and runs it
   through a filter that converts the sample rate.


Required files and media
========================
   Copy the media tree to the target machine before running this sample.


Programming Notes
=================
   The sample rate converter is a sample XMO. It's purpose here is just to be
   simple.
