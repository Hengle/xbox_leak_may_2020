Description
===========
   Demonstrates how to use DirectMusic 3D AudioPaths to play sound effects
positioned in 3D space.  The sample has the same functionality as the 
Play3DSound sample, but does it using DirectMusic instead of DirectSound.


Required files and media
========================
   Copy the media tree to the target machine before running this sample.
